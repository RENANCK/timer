# Hiperfoco em Medicina

Aplicativo estático em HTML/CSS/JS com cronômetro oculto para estímulos aleatórios.

## Funcionalidades

- Ativação de ciclos aleatórios ocultos entre **10 e 45 minutos**.
- Exibição apenas de mensagens de status (sem revelar o tempo sorteado).
- Alarme em loop ao final do ciclo usando `assets/alarme.mp3`.
- Botão **Parar música e continuar** para iniciar novo ciclo oculto.
- Botão **Desativar** para interromper timer, alarme e voltar ao estado inicial.

## Arquivos

- `index.html`
- `style.css`
- `script.js`
- `assets/alarme.mp3` (adicione manualmente este arquivo local)

## Como usar

1. Coloque seu áudio de alarme em `assets/alarme.mp3`.
2. Abra `index.html` no navegador.
3. Clique em **Ativar** para iniciar.
